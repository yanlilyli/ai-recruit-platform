# 光子S工作室 · AI招聘交叉面试派单系统

> 单文件HTML应用，零依赖，可直接部署到任何静态托管。

## 在线访问

部署到 GitHub Pages 后访问：`https://yanlilyli.github.io/ai-recruit-platform/`

## 角色权限

- **📋 需求方下单**：所有人可填写岗位的AI能力需求
- **📐 能力标尺**：所有人可查看 L0/L1/L2/L3 评估标准
- **🔐 管理员后台**：仅招聘负责人可访问（需密码）

## 能力标尺

| 等级 | 名称 | 定义 |
|------|------|------|
| L0 | 无需具备 | 该能力对岗位非核心要求 |
| L1 | 了解（应用调用级） | 能调用现成工具，跑通Demo |
| L2 | 熟练（业务落地级） | 能微调+工程化，独立完成业务集成 |
| L3 | 精通（算法与架构级） | 深入理解算法架构，能改进模型/优化推理 |

## 数据存储

当前为前端 localStorage 存储（每个浏览器独立）。如需多人共享数据，可接后端API。

## 技术栈

- 纯HTML/CSS/JavaScript，零依赖
- 数据存储：localStorage / sessionStorage
- 派单算法：必备项L级覆盖度 75% + 加分项 25%
